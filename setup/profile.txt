==== Perf ====
* Install linux-tools-generic
* You can also download source code of linux kernel, and run "make" in tools/perf
* To check all the available performance counters, run
perf list
* To collect performance counters every one second (1000ms), run
perf stat -e xxx,xxx,xxx -p <pid> (or -c <cores>) -g -I 1000


==== Systemtap ====
* Tutorial: http://www.redbooks.ibm.com/redpapers/pdfs/redp4469.pdf
* To intall on the Cavium board, you'll have to download the source code of systemtap (don't use apt-get install), and update linux kernel to >=4.8


==== Flamegraph ==== 
* Source code and intro: https://github.com/brendangregg/FlameGraph
* Flamegraph can help you identify call stack of an application and present it in a neat way
* Tools to generate stack traces
  * perf record 
  * sample-bt in systemtap-toolkit: https://github.com/openresty/openresty-systemtap-toolkit


==== Top ====
Linux top helps you monitor cpu utilization. To dump real-time top results to a file, run
    top -p <pid> -d <refresh duration in second> -bn<#iterations> > filename
