==== NGINX ====
1. Install
  sudo apt-get -y install build-essential libssl-dev libpcre3-dev
  wget http://nginx.org/download/nginx-1.12.0.tar.gz
  tar xf nginx-1.12.0.tar.gz
  cd nginx-1.12.0
  sudo ./configure
  sudo make -j
  sudo make install
2. Read and understand the source code. Here is two useful Chinese blogs (sorry it's in Chinese): 
    https://www.cnblogs.com/muhy/p/10528543.html
    https://blog.csdn.net/weixin_43844810/article/details/90639433
3. Configure NGINX to boost performance
  * Here is a useful link: http://nginx.org/en/docs/beginners_guide.html
  * Configure CPU affinity: "worker_cpu_affinity" in nginx.conf
  * Configure thread pool to make syscalls non blocking: http://nginx.org/en/docs/ngx_core_module.html?&_ga=1.68296344.843168382.1477489692%23accept_mutex
  * Open file cache to cache file descriptors (but this seems outdated. It may not be necessary for newer versions of NGINX): https://serverfault.com/questions/460275/how-to-choose-a-correct-value-for-open-file-cache-in-nginx-configuration
  * Too many open files? Add “worker_rlimit_nofile number” in nginx.conf: https://gist.github.com/joewiz/4c39c9d061cf608cb62b
  * Another useful link: https://www.slashroot.in/nginx-web-server-performance-tuning-how-to-do-it
  * Change "ulimit -n"


==== Memcached ====
Install:
  sudo apt-get build-dep memcached -y
  wget http://memcached.org/files/memcached-1.4.36.tar.gz
  tar xf memcached-1.4.36.tar.gz
  cd memcached-1.4.36
  ./configure
  make
  sudo make install
